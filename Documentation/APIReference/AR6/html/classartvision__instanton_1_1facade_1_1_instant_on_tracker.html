Parameters
dataPointer to unpadded 8-bit luminance image.
widthWidth in pixels of image pointed to by data.
heightWidth in pixels of image pointed to by data.
timestampTime at which the image was acquired in decimal seconds. The timebase for timestamp is system-dependent. On iOS and macOS, it is the same time base used for CoreMotion data, which is the elapsed time since system start. This is also the timebase used by ARVideo's AVFoundation module, and the same as the timebase used by CoreAnimation e.g. via the function CACurrentMediaTime() (in QuartzCore.framework).
