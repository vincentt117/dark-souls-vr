Detailed Description

Implements a track index for maintaining links between feature point observations in a video sequence.

It can manage an arbitrary number of tracks. The number of frames to manage can be restricted for memory utilization. The correspondences between any pair of frames can be queried.

Get the correspondences between two frames in the frame buffer.

The time complexity is linear with the number of tracks at frame (t-a).

Parameters
[out]paPoint for frame (t-a)
[out]pbPoint for frame (t-b)
[in]aRelative index of frame (t-a)
[in]bRelative index of frame (t-b)

The two frames are indexed by the variables (a,b). The most recent (current) frame "t" is at index 0. The variables (a,b) are relative to the current frame so that (t-a,t-b) are the two frames where correspondences are requested such that a < b.
